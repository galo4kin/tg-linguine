# 54 — Подключить XP/стрик/дневную цель к квизу + сводка раунда

## Цель
Связать хранилище из шага 53 с хендлером квиза: показывать прогресс в фидбэке и сводке, начислять XP и обновлять стрик.

## Что сделать
- В хендлере квиза перед стартом раунда: `progress.RolloverIfNewDay(ctx, userID, today)`.
- На каждом правильном ответе: `progress.RecordCorrect(...)`. Использовать возврат:
  - если `GoalJustHit` — добавить в feedback-сообщение строку «🎯 Дневная цель достигнута! +N XP» (где N = `QUIZ_XP_BONUS_GOAL`).
  - всегда добавлять в feedback `+10 XP`.
- На неправильном ответе: `progress.RecordWrong` (стрик дней не трогаем).
- Расширить summary раунда (из шага 50): `Правильно X/10 • +M XP за раунд • Стрик: K 🔥 • Цель: P/Q`.
- Команда `/me` (или расширение существующей, если есть `/profile`): показывает уровень/язык, XP, стрик, цель. Если такой команды ещё нет — добавить минимальную в `internal/telegram/handlers/`.
- i18n: `quiz.feedback.xp`, `quiz.feedback.goal_hit`, `quiz.summary.xp`, `quiz.summary.streak`, `quiz.summary.goal`, `me.title`, `me.xp`, `me.streak`, `me.goal` (en/es/ru).

## Definition of Done
- Дойти до 20 правильных ответов за день → увидеть бонус ровно один раз; повторное прохождение в тот же день не добавляет бонус.
- Правильный ответ на следующий день → стрик +1; пропуск дня → стрик сбрасывается до 1 при следующем правильном ответе.
- `/me` показывает корректные значения.
- `make build` (тесты по возможности), бот перезапущен.
- Один коммит `step 54: quiz-progress-and-summary`.
