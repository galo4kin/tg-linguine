# Шаг 20. Settings menu (3–4h)

## Цель
Меню `/settings` — управление языком интерфейса, активным языком
изучения, уровнем CEFR, API-ключом, удаление данных.

## Действия
1. Команда `/settings` → меню с пунктами:
   - «Язык интерфейса» (ru/en/es) → обновляет `users.interface_
     language`,
   - «Изучаемый язык» → выбор из существующих или добавить новый
     (вернуться в onboarding-стиль для нового),
   - «Уровень CEFR» → A1..C2, обновляет `user_languages.cefr_level`
     для активного языка,
   - «API-ключ» → переписать (повторно валидируется),
   - «Удалить мои данные» → переход к шагу 24 (заглушка пока, если
     не реализован).
2. Все изменения применяются мгновенно. После смены интерфейсного
   языка ответ бота сразу на новом языке.
3. i18n покрытие всех новых строк.

## DoD
- Все настройки меняются и сохраняются.
- Смена интерфейсного языка отражается в следующем же ответе.
- Смена CEFR влияет на адаптацию (шаг 19).
- `make build` + `make test` зелёные.

## Зависимости
Шаги 8, 19.
