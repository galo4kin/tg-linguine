# Шаг 27. Базовая безопасность контента (2–3h)

## Цель
Блокировка adult/illegal: статический blocklist доменов + проверка
`safety_flags` от LLM.

## Действия
1. `configs/blocked_domains.txt` — по домену в строке, комментарии
   `#`. Подгружается через `embed`.
2. Проверка домена URL до извлечения. Hit → отказ, без сетевого
   запроса.
3. После LLM-ответа: если `safety_flags` непустой → не сохраняем
   статью, отвечаем сообщением.
4. i18n-сообщения: «Этот источник не поддерживается» и «Контент
   заблокирован».

## DoD
- URL из blocklist → отказ без сетевого запроса (verify через лог
  `extractor_called=false`).
- LLM-флаг → запись не появляется в `articles`.
- Юнит-тест на blocklist matching (поддомены тоже блокируются).
- `make build` + `make test` зелёные.
