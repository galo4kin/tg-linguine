# Рефактор. Убрать мёртвый метод (*sqliteRepo).WithTx (15min)

## Цель
Удалить неиспользуемый метод `(*sqliteRepo) WithTx(ctx, fn) error` из
`internal/articles/repo.go`.

## Контекст
В пакете `internal/articles` сейчас есть две сущности с именем
`WithTx`:

- свободная функция `articles.WithTx(ctx, db, fn) error` — её
  используют `usecase.go`, `repo_test.go`,
  `dictionary/repo_test.go`;
- метод-обёртка на `*sqliteRepo`, идентичная по семантике, нигде не
  вызывается.

Метод не входит в интерфейс `Repository`, не экспортируется через
него, и просто шумит.

## Действия
1. В `internal/articles/repo.go` удалить блок:
   ```go
   func (r *sqliteRepo) WithTx(ctx context.Context, fn func(tx *sql.Tx) error) error {
       return WithTx(ctx, r.db, fn)
   }
   ```
2. Прогнать `make build` и `make test`.

## DoD
- Метод удалён.
- Файл компилируется, все тесты зелёные.
- Семантика свободной функции `articles.WithTx` не меняется.

## Зависимости
Шаг 15.
